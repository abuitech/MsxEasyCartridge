# Version 0.33
## Installation
After updating the Easy Cartridge firmware with **BasicRomEmulator_v0.33.uf2**, copying the folder **.sys** into Easy Cartridge at the root folder.
## Changes in this release
1. Easy Cartridge uses a database (exported from blueMSX database) to find the rom type before trying to guess it.
2. Support Konami Gamemaster 2 cartridge (with SRAM save on the cartridge flash memory)
3. Support of the following mapper :
    * Konami Generic
    * Ascii 16 Generic
    * Ascii 8 SRAM
    * Ascii 16 SRAM
